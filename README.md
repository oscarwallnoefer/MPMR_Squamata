# MPMR_Squamata

### Goal

Develop a workflow from Orthofinder to TreePuzzle on the whole proteome. 

### Dataset

Same-size groups (~8 species per group). 
Groups:

+ Agamidae
+ Pleurodonta
+ Serpentes
+ Others

### Steps

0. Genome quick annotation
1. Orthofinder
2. IQ-TREE2
3. ERCnet
